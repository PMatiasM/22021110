## Getting Started

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/PMatiasM/our-map-api.git
```

2. Install NPM packages

```sh
npm install
```

3. Enter your configs in `.env`

```JS
MONGO_DATABASE=
MONGO_URL=

SERVER_PORT=
```

4. Have fun!
